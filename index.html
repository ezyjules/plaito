AI Study Buddy — what it is & how to start

A friendly tutor that reads a photo of a homework question and works through it with you — checking your answer step by step, or coaching you when you're stuck, with hints before answers. Any subject, any year level. This is a beta and can occasionally make a mistake, so always sanity-check its arithmetic.

Honestly, it's easier than it looks. "API key" sounds technical, but the whole thing is just: make an account, add a few dollars, click Create key, and copy-paste a long password into this app. No coding, no installing anything. Most parents are done in about five minutes — and you only do it once per device. The links below each open in a new tab, so keep this page open to paste back into.
Is it safe? Yes — there's nothing to leak. There's no company server and no account. Everything — your key, your settings, your progress — is stored only in this browser on this one device. Nothing is ever sent anywhere except directly to the AI provider you choose, to answer each question. Because nothing is collected centrally, there's no pool of data to breach.
One device at a time. Because it all lives in this browser, you'll do this quick setup once on each device you want to use — the laptop, the iPad, a phone. They don't share. Two minutes each, then it's remembered until you clear your browsing data.

What's an API key, and why do I need one?

The "tutoring brain" is rented directly from an AI company. An API key is like a prepaid SIM card for AI: you load a little credit, and each question sips about a cent of it. There's no subscription. $5–10 of credit typically lasts a whole school term, and you can watch exactly what's spent on the provider's website. Pick one of the two below.

Option A — Claude key (Anthropic)

  1. Open console.anthropic.com (opens in a new tab — keep this one open to paste into) and sign in. You can use an existing Claude login.
  2. Open Billing and add a small amount of credit (e.g. $5–10). Leave auto-reload off so it can never run away.
  3. Open API Keys → Create Key. Name it study-buddy.
  4. Copy the key (it starts with sk-ant-) — it's shown only once.
  5. Back here: in setup, choose Claude (Anthropic), paste the key, and finish. Done.

→ Open the Anthropic console in a new tab

Option B — ChatGPT key (OpenAI)

  1. Open platform.openai.com (new tab) and sign in.
  2. Open Billing and add a little credit ($5–10), auto-reload off.
  3. Open API keys → Create new secret key. Name it study-buddy. Default (unrestricted) permissions are fine.
  4. Copy the key (it starts with sk-) — shown only once.
  5. Back here: in setup, choose ChatGPT (OpenAI), paste the key, and finish.

→ Open the OpenAI keys page in a new tab

Nicer voice (optional) — ElevenLabs

The tutor reads aloud using the free voices built into your device. They're fine, but a little robotic. ElevenLabs makes startlingly human AI voices, and it's optional and free to start.

  1. Sign up at elevenlabs.io (new tab — the free tier covers roughly 10 minutes of speech a month, plenty to try).
  2. Click your profile (bottom-left) → API Keys → Create API Key. Name it study-buddy.
  3. If it offers permissions, enable Text to Speech and Voices (read) — or just leave it unrestricted. A restricted key is the usual reason a key gets rejected.
  4. Copy the key (it starts with sk_).
  5. Back here: paste it into the Premium voice box in setup and click Connect voices. Starred ⭐ voices will appear in the voice list.

→ Open the ElevenLabs API keys page in a new tab

Mac tip — free upgrade: even without ElevenLabs, you can download much better system voices at System Settings → Accessibility → Spoken Content → Manage Voices (look for "Premium" English voices). They'll show up in the voice list automatically.

Keeping your keys safe

  • A key only ever belongs in this app's setup box — never paste it into an email, message, or chat.
  • If a key is ever exposed, revoke it on the provider's site (takes seconds) and create a new one. The most you can lose is the small credit you loaded.

Tips for good results

  • Photograph one question at a time, or snap the page and drag a box around one question.
  • Clear handwriting helps the tutor read your working — neat is accurate.
  • Checking an answer? Make sure your working is inside the highlighted box.
  • If an answer looks wrong, tell the tutor in the chat — it'll recheck, and that's half the learning.
